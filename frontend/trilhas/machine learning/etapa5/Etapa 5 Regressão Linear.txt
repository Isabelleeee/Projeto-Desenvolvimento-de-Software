Etapa 5: Regressão Linear
Resumo:
 Usada para prever valores contínuos (valores numéricos). O mecanismo principal é ajustar uma linha aos dados, desse modo minimizando o erro entre os valores do treino e os reais. 
A equação que define a regressão linear é: 
y = mx + b
O melhor método é definir bons valores para m e b (vetor gradiente), minimizando erros
