Etapa 3: Conceitos Básicos de Dados
Resumo: A estrutura dos básica dos dados é definida por samples (cada sample é uma linha do database), features (colunas do database) e o target (o que queremos prever). 
A dimensionalidade dos dados é determinada a complexidade e/ou ao numero de variaveis de entrada, dependente das features.
